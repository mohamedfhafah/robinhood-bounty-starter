# Out-of-scope issue classes (non-exhaustive)

- Physical attacks, social engineering, phishing.
- Most informative-only findings (example: SPF/DNS policy-only issues).
- DoS without prior authorization.
- Subdomain takeover without takeover proof.
- Cache poisoning.
- Clickjacking without security impact.
- Disclosure of publicly available information.
- Missing cookie flags except session cookies.
- Missing security headers unless exploitable.
- TLS/cipher findings unless exploitable.
- Logout CSRF.
- Mobile findings requiring rooted/jailbroken devices.
- Third-party integration issues that do not demonstrate exploitability against Robinhood.
